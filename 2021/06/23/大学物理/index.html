狭义相对论

洛伦兹变换

$$x^{‘} = \gamma (x - ut)\ \gamma = \frac{1}{\sqrt{1-\frac{u^2}{c^2} } }\t^{‘} = \gamma(t - \frac{ux}{c^2})\速度变换：v_x^{‘} = \frac{v_x - u}{1-\frac{uv_x}{c^2}}\v_y^{‘}=\frac{v_y}{1-\frac{v_yu}{c^2}}\frac{1}{\gamma}$$

洛伦兹变换不变量： 洛伦兹协变矢量四分量平方和

$$\begin{bmatrix}x\y\z\ict\end{bmatrix}$$

—则洛伦兹变换实际上相当于洛伦兹协变量在四维空间的旋转变化

任意两事件在不同参考系下的洛伦兹协变间隔相同

以光信号相互影响的两事件 洛伦兹协变量之差 分量平方和为0

相对论下能量与动量

$$E^2=p^2c^2+m_0^2c^4\E=mc^2=c^2m_0\gamma\p = m_0v\gamma\F=\frac{dp}{dt}$$

电子静止能量$m_0c^2=0.51\times 10^6eV$

刚体转动

基本公式

$$v = \omega \times r\M=r \times F\\alpha = d\omega/dt\I = \sum \Delta m_ir_i^2\M= I\cdot\alpha = dL/dt\L = mr\times v\L=I\cdot w\E_k = \frac{1}{2}\omega^2 m $$

质心系下以质心为轴：非惯性力合力矩为0 故不必考虑非惯性力也有

$$M_{外}=dL_{c}/dt$$

天体运动

轨道能量$E=-\frac{GMm}{2a}=-\frac{GMm}{r}+\frac{1}{2}mv^2$

角动量$L=mr^2\frac{d\theta}{dt}=m(a+c)v_{a+c}=m(a-c)v_{a-c}$

距离地心$r_{max}=a+c,r_{min}=a-c$

有心力场运动公式 $\frac{1}{2}m(\frac{dr}{dt})^2+\frac{L}{2mr^2}+V(r)=E$

化简得: $\frac{1}{2}\frac{L^2}{mr^4}(\frac{d\theta}{dt})^2+\frac{L}{2mr^2}+V(r)=E$

由此式解微分方程可得有心力场下物体的运动轨迹方程$r=r(\theta)$

振动

简谐运动

$$\frac{d^2x}{dt^2}+\omega^2x=0\w=\sqrt{\frac{k}{m} }\E=\frac{1}{2}kA^2=E_p+E_k$$

阻尼振动

基本公式

$$f=-\gamma\frac{dx}{dt}\m\frac{d^2x}{dt^2}=-kx-\gamma\frac{dx}{dt}\\omega_0=\sqrt\frac{k}{m},2\beta=\frac{\gamma}{m}\\frac{d^2x}{dt^2}+2\beta\frac{dx}{dt}+w_0^2x=0$$

运动方程解为：

$$x=A_0e^{-\beta t}cos(\omega t +\varphi_0)\\omega=\sqrt{w_0^2-\beta^2}\T=\frac{2\pi}{\omega}$$

由此知阻尼振动周期相比无阻尼变长

受迫振动

同时考虑驱动力和阻力

模型

$$F_{驱动}=Hcos(\omega t)\ 令\omega_0=\sqrt\frac{k}{m},2\beta=\frac{\gamma}{m},h=\frac{H}{m}\则运动方程\frac{d^2x}{dt^2}+2\beta\frac{dx}{dt}+w_0^2x=hcos\omega t\微分方程解为x=A_0e^{-\beta t}cos(\sqrt{\omega_0^2-\beta^2}t+\varphi_0)+Acos(\omega t+\varphi)\ 达到稳定状态时上式第一项忽略\振幅A=\frac{h}{[(\omega^2_0-\omega^2)^2+4\beta^2\omega^2]^{\frac{1}{2}}}\ 稳态受迫振动与驱动力相位差\varphi=arctan\frac{-2\beta\omega}{\omega^2_0-\omega^2}\ \omega_r=\sqrt{\omega_0^2-2\beta^2}时达最大振幅$$

同一直线上同频率振动合成

$$x_1=A_1cos(\omega t+\varphi_1)\x_2=A_2cos(\omega t+\varphi_2)\合成振幅：A=\sqrt{A_1^2+A_2^2+2A_1A_2cos(\varphi_2-\varphi_1)}\ 合成振动初相位 tan\varphi=\frac{A_1sin\varphi_1+A_2sin\varphi_2}{A_1cos\varphi_1+A_2cos\varphi_2} $$

不同频率合成

$$x_1+x_2=2Acos(\frac{\omega_2-\omega_1}{2}t)cos(\frac{\omega_2+\omega_1}{2}+\varphi)\某时刻t最大增幅2Acos\frac{\omega_2-\omega_1}{2}t\拍频\nu=\nu_2-\nu_1，为两分振动频率之差。\指振幅在单位时间内出现最大振幅的次数$$
